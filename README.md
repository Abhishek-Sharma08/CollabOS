CollabOS is a **production-grade, full-stack collaboration platform** built for modern dev teams. It's not just a project manager ‚ÄĒ it's the intelligence layer that sits between your team, your code, and your tools.

- ūü§Ė Let **Gemini AI** plan your entire project from a single description
- ūüĒī See which teammates are **online in real-time** via Socket.io presence
- ūüĒó Sync **GitHub commits** directly into your project timeline
- ūüďĀ Link **Google Drive folders & Docs** to any project

One system. Zero context-switching.

## ūüüĘ Real-Time Presence ‚ÄĒ Socket.io

One of CollabOS's standout features is **live team presence**. Every authenticated user emits their status to the server when they connect, and all teammates can instantly see who's online ‚ÄĒ no polling, no refresh.

**How it works:**

```
User Opens CollabOS
       ‚Üď
   JWT Verified
       ‚Üď
 Socket.io Handshake
       ‚Üď
Server registers user as "online"
       ‚Üď
Broadcasts to all team members
       ‚Üď
ūüüĘ Avatar goes live in dashboard
       ‚Üď
On disconnect ‚Üí status removed, team notified
```

**Events:**
| Event | Direction | Description |
|---|---|---|
| `user:online` | Client ‚Üí Server | User connected |
| `user:offline` | Server ‚Üí Client | User disconnected |
| `team:presence` | Server ‚Üí Client | Full list of online team members |
| `task:updated` | Server ‚Üí Client | Real-time task status change |

## ūüĒĎ Google OAuth Flow

Users can authenticate with their Google account instead of (or in addition to) email/password.

```
User clicks "Sign in with Google"
           ‚Üď
  Redirected to Google Consent Screen
           ‚Üď
  Google returns auth code
           ‚Üď
  Backend exchanges code ‚Üí access + refresh tokens
           ‚Üď
  User record created / updated in MongoDB
           ‚Üď
  JWT issued ‚Üí user is logged in
           ‚Üď
  Google tokens stored for Drive/Docs integration
```

This same OAuth session is also reused for **Google Drive** and **Google Docs** access ‚ÄĒ no re-authentication needed.

## ūüöÄ Getting Started

### Prerequisites

- Node.js v18+
- MongoDB (local or Atlas)
- Google Cloud project (OAuth + Drive API enabled)
- GitHub OAuth App
- Gemini API key

### 1. Clone

```bash
git clone <repo-url>
cd collabos-backend
```

### 2. Install

```bash
npm install
```

### 3. Configure `.env`

```bash
cp .env.example .env
```

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
CLIENT_URL=http://localhost:5173

# Google AI
GEMINI_API_KEY=your_gemini_api_key

# GitHub OAuth
GITHUB_CLIENT_ID=your_github_client_id
GITHUB_CLIENT_SECRET=your_github_client_secret

# Google OAuth + Workspace
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
GOOGLE_REDIRECT_URI=http://localhost:5000/api/auth/google/callback
```

### 4. Run

```bash
npm run dev
```

API: `http://localhost:5000`  
Socket.io: same port, auto-upgraded
